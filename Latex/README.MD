Задание 1:
 В рамках данного задания необходимо преобразовать текст своего варианта, представленный в файле .txt, в формат LaTeX, оформив его в соответствии с образцом, приведённым в .pdf .

.txt файлы содержат в себе текст в кодировке Windows-1251. Изучите команду `iconv`, чтобы изменить кодировку на UTF-8.

Требования к обработке текста

Исходный текст представляет собой набор рассказов. 

Необходимо разбить текст на абзацы (добавить пустые строки между абзацами), удалить разрывы слов (переносы), привести пунктуацию к стандартам LATEX
 для русского языка (кавычки: << >>; тире: "--- , "--~ , "--* (в зависимости от контекста); дефисы: "=; многоточие: \dots)
 
Каждый рассказ (без заголовка) должен быть вынесен в отдельный .tex-файл.
Все рассказы должны подключаться в основные документы с помощью команды \input. Текст рассказов не должен дублироваться в основных файлах.
Формирование двух документов


Первый документ должен быть типа article и иметь название «Избранные произведения». В нём каждый рассказ оформляется как отдельная секция с использованием команды \section, причём сама команда \section должна находиться в основном файле, а текст рассказа подключаться из внешнего файла с помощью команды \input.

Второй документ должен быть типа book и также иметь название «Избранные произведения». В этом документе каждый рассказ оформляется как отдельная глава с использованием команды \chapter, которая должна находиться в основном файле, тогда как текст рассказа подключается через \input. При этом должны использоваться те же самые файлы с текстами рассказов, что и в документе типа article.
